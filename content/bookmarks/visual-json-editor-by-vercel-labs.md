visual-json, JSON düzenlemeyi daha kullanıcı dostu hale getiren bir araç seti sunuyor.

Öne çıkan tarafları:

- Schema-aware yapı ile JSON şemasıyla uyumlu düzenleme akışları
- Embeddable ve extensible yaklaşım (ürüne entegre etmeye uygun)
- Paket ayrımı:
  - `@visual-json/core`: headless tree model ve operasyonlar
  - `@visual-json/react`: TreeView/FormView/DiffView dahil React UI bileşenleri
  - `@visual-json/vscode`: görsel JSON editörü uzantısı

Resmi site: https://visual-json.dev
