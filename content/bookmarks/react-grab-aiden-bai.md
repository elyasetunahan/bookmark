Aiden Bai tarafından duyurulan React Grab, çalışan bir sayfada herhangi bir elementi görsel olarak seçip Claude Code veya OpenAI Codex'e doğrudan değişiklik talimatı vermeyi mümkün kılıyor.

Öne çıkan kullanım alanları:

- Canlı sayfada bir bileşeni tıklayıp doğal dille yeniden tasarlamak veya refactor etmek
- Claude Code / Codex entegrasyonuyla AI destekli in-browser düzenleme akışı kurmak
- `npx react-grab@latest` ile mevcut projeye kurulum gerektirmeden denemek
